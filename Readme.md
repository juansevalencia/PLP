# Paradigmas de la programación - PLP

En Paradigmas de la Programación aprendí a ver que la forma en que vos modeles tu mundo dentro de la programación puede dificultar o facilitar el camino para encontrar la solución a los problemas. 
Abarque los siguientes paradigmas:
 
##1. Funcional
Que por ejemplo puede servir mucho mas para resolver problemas matematicos ya que todo es una función (Haskell) 

##2. Lógico.
Aquí usamos un lenguaje llamado calculo lambda.
 Donde todo es un predicado. 
Al trabajar con esto tenemos una base muy fuerte para demostrar correctitud mediante la logica, induccion estructural que le da una formalidad muy importante.
Haber tenido que pasar al paradigma Lógico después de nunca haberlo visto y trabajar con otros paradigmas fue un desafio muy grande, cambiar de paradigma, cambiar la forma de resolver las cosas hizo que cambie la dificultad de muchos problemas a comparación de otros paradigmas. Fue muy pesada ya que devuelta teníamos pocos profesores para la cantidad de alumnos y guias semanales de 30 ejercicios.
